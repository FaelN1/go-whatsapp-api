=====================================================
     COMO EXECUTAR AS MIGRATIONS - 3 MINUTOS
=====================================================

📍 VOCÊ ESTÁ AQUI:
   d:\work\go-whatsapp-api

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎯 MÉTODO RÁPIDO - SCRIPT AUTOMÁTICO:

   cd d:\work\go-whatsapp-api
   scripts\windows\migrate.bat

   → O script vai pedir confirmação antes de executar

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔧 MÉTODO MANUAL - COMANDO ÚNICO:

   1. Abra PowerShell nesta pasta
   
   2. Execute:
   
      psql "postgresql://postgres:suasenha@localhost:5432/whatsapp_api" -f internal\platform\database\migrations\001_create_analytics_tables.sql

   3. Substitua:
      - postgres: seu usuário
      - suasenha: sua senha
      - localhost: seu host (se diferente)
      - whatsapp_api: nome do seu banco

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖱️ MÉTODO VISUAL - INTERFACE GRÁFICA:

   1. Abra pgAdmin ou DBeaver
   
   2. Conecte ao banco "whatsapp_api"
   
   3. Abra este arquivo:
      internal\platform\database\migrations\001_create_analytics_tables.sql
   
   4. Execute (F5 ou botão Run)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ VERIFICAR SE DEU CERTO:

   Conecte ao banco e execute:
   
   \dt message_*
   
   Deve mostrar:
   - message_tracking
   - message_views
   - message_reactions

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ PROBLEMAS COMUNS:

   "psql not found"
   → Instale PostgreSQL client
   → Windows: https://www.postgresql.org/download/windows/

   "database does not exist"
   → Crie primeiro: CREATE DATABASE whatsapp_api;

   "permission denied"
   → Verifique usuário/senha no comando
